# LUFE ART TATTOO

A cinematic luxury tattoo studio website built with modern web technologies.

Designed to feel immersive, artistic and premium — inspired by luxury editorials, underground art galleries and cinematic experiences.

---

## ✨ Experience

LUFE ART TATTOO is not a traditional tattoo studio website.

This project focuses on:
- cinematic storytelling
- immersive UI/UX
- luxury visual direction
- premium motion design
- editorial layouts
- responsive performance
- high-end branding

The experience is optimized for both desktop and mobile while preserving visual identity and performance.

---

# 🖤 Tech Stack

- Next.js 14
- TypeScript
- TailwindCSS
- Framer Motion
- GSAP
- Lenis Smooth Scroll
- React Three Fiber
- Three.js

# 🎨 Features

## Cinematic UI
- dark luxury aesthetic
- immersive hero section
- premium typography system
- editorial layouts
- cinematic transitions
- subtle ambient motion

## 3D Logo Experience
Desktop-only interactive 3D logo experience using React Three Fiber.

Features:
- cinematic lighting
- floating motion
- emissive crack glow
- smooth interactions

Mobile devices automatically fallback to an optimized flat version for performance.

## Premium Portfolio
- editorial gallery layout
- immersive fullscreen previews
- category filtering
- cinematic image transitions

## Booking Experience
Luxury multi-step booking flow integrated with WhatsApp.

## SEO Optimized

- robots.txt
- structured data
- semantic HTML

## Fully Responsive
Optimized for:
- mobile
- tablets
- desktop
- ultrawide displays

# 🚀 Getting Started

## 1. Install dependencies

```bash
npm install
```

---

## 2. Run development server

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```

---

## 3. Production build

```bash
npm run build
```

---

## 4. Start production server

```bash
npm run start
```
